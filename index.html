Featured Work

RL controller for the Kessler Game simulation environment

Built a hybrid controller that combines Sugeno-type fuzzy logic, PPO, scenario conditioning, and staged training across multiple scenario groups.

Python PyTorch PPO + GAE Fuzzy Logic
  • Staged trainingUsed foundation, motion, pressure, and full scenario groups to build the controller progressively.
  • 50% faster trainingUsed GAE, mini-batch updates, and KL-based stopping to cut training time while maintaining performance.
  • Context inputsUsed time-to-collision, relative velocity, and spatial signals to improve control decisions.
