Featured Work

Hybrid RL controller for Kessler Game

Built a hybrid control system that combines Sugeno-type fuzzy logic with PPO reinforcement learning, using a shared-trunk policy to coordinate movement and combat decisions in a single forward pass.

Python PyTorch PPO + GAE Fuzzy Logic
  • Shared-trunk policyUnified thrust, turning, firing, and mine actions to improve coordination across control tasks.
  • 50% faster trainingUsed GAE, mini-batch updates, and KL-based stopping to cut training time while maintaining performance.
  • Scenario-conditioned policyImproved decision quality with time-to-collision, relative velocity, and spatial prioritization signals.
